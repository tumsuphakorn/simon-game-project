Simon Game

One basic rule for this game... remember the sequence and repeat it correctly!


  1. Select difficulty level:
    • Easy (4 boxes)
    • Medium (9 boxes)
    • Hard (16 boxes)
  2. Press start and be prepared for oncoming flashing boxes
  3. Remember the sequence and repeat it
  4. The sequence will be longer by 1 box after each turn
  5. That means you need to remember longer and longer sequence
  6. Still don't understand... let's try!
